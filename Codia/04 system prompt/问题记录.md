### 环境信息如何注入 System Prompt

**问：** 当前系统，环境信息是怎么加入到 system prompt 里面的？

**答：** 在 `ChatService` 构造时，`buildEnvInfo()` 收集 OS/Shell/日期/工作目录/Git分支，作为纯文本拼接到 system prompt 末尾。关键文件：

- `src/chat/chat-service.ts:638-659` — `buildEnvInfo()` 收集环境信息
- `src/chat/chat-service.ts:277-298` — base prompt + envInfo 拼接
- `src/prompt/builder.ts` — SystemPromptBuilder 按 priority 组装 Section
- `src/prompt/reminders.ts:21-77` — `createEnvInfoProvider()` 也用类似收集逻辑但包装成 `<system-reminder>` user message，目前在 ChatService 中未被使用

数据流：`ChatService.create() → buildNewSessionContext() → ChatService 构造函数 → SystemPromptBuilder.add(固定Section) → build() → + buildEnvInfo() → fullSystemPrompt`
