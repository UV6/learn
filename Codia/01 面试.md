## 项目介绍

轻量级 Coding Agent，基于 ReAct 架构驱动 LLM 自主完成编程任务，五层分层架构支持多模型接入、MCP 工具扩展与多 Agent 协作

## 个人职责

- 独立负责五层分层架构设计与全部核心技术决策，借助 Claude Code 完成模块编码与调试
- 设计并实现跨会话记忆系统，包括 JSONL 持久化、LLM 自动记忆提取与项目指令文件分级加载
- 搭建 Agent 评测体系，基于 SWE-bench-Live 数据集做自动化评测，设计上下文信息保留测试，通过评测数据驱动 prompt 优化

## 技术亮点

- MCP 工具延迟加载：设计 MCP 工具延迟加载机制，百级工具场景下工具描述 Token 占用减少 85%，避免上下文窗口被工具定义挤占
- 两层上下文压缩：设计两层渐进式上下文压缩策略，支持 8 小时以上连续编程会话不丢失关键上下文
- 评测驱动优化：基于 SWE-bench-Live 数据集搭建评测流水线，验证 Agent 的 bug 定位与修复能力，自主设计上下文信息保留测试，发现并修复压缩摘要模板缺陷，信息保留率从 17% 提升至 100%
- 五层安全隔离：设计五层权限确认机制，将单次会话权限弹窗从平均 30 次降至 5 次以内，避免审批疲劳影响使用体验；可选用 OS 级沙箱进一步兜底，内核级隔离支持 Agent 全自动模式安全执行
- 多 Agent 协作：设计多 Agent 并行协作与文件级隔离机制，突破单 Agent 上下文窗口瓶颈，跨文件重构任务相比单 Agent 耗时降低约 60%
-  Agent 工作流标准化：将测试驱动开发与规范驱动开发固化为内置 Skill，用户通过 /tdd、/sdd 一键触发 red-green-refactor 循环与 spec-plan-task-checklist 文档生成，降低高质量编码与需求规划的操作门槛，沉淀可复用的 Agent 标准作业程序。
