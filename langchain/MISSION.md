# Mission: LangChain + LangGraph for Agent Development

## Why
从 Java 后端转型 AI Agent 应用开发，需要在 1-2 周内掌握 LangChain 和 LangGraph 两大框架的核心能力，达到能通过 agent 开发岗位面试并上手实际项目的水平。

## Success looks like
- 能独立搭建一个完整的 RAG（检索增强生成）应用，包括文档加载、切片、向量存储、检索链
- 能设计和实现一个基于 LangGraph 的多步骤 Agent 工作流，包含工具调用、条件分支、状态管理
- 能在面试中清晰解释 Agent 的核心概念：Chain vs Agent、Tool/Function Calling、Memory、多 Agent 协作模式
- 有一个可展示的 GitHub 项目（端到端的智能问答或自动化 Agent）

## Constraints
- 时间：1-2 周（紧迫）
- 语言：Python（目前仅了解语法，但 Java 基础深厚）
- 学习方式：通过 Codia 助手进行交互式学习，需要高密度、实战导向的内容

## Out of scope
- Python 语言基础教程（已有语法知识，边用边查）
- 大模型训练/微调（专注应用层开发）
- LlamaIndex 等替代框架（聚焦 LangChain 生态）
- MLOps、模型部署、向量数据库运维（面试后再学）
