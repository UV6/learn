# Prior Knowledge Baseline

用户在开始学习时已具备：Java 后端开发经验、Python 语法基础了解、Transformer 架构知识。这意味着：
- 编程概念（面向对象、类型系统、API 设计）不需要重新教
- Python 语法可边用边查，不需要单独课程
- LLM 工作原理（token、context window、temperature）有基础认知

**Implications**: 课程可以直接从框架应用层开始，跳过语言基础和 AI 原理。重点放在 LangChain/LangGraph 的特有抽象上：Agent Loop、Tool 定义、Graph 编排。
