# LangChain + LangGraph Resources

## Knowledge

- [LangChain Python 官方文档](https://docs.langchain.com/oss/python/langchain)
  **核心参考。** 覆盖 Agent 创建 (create_agent)、RAG、工具集成、Memory、Streaming。作为日常开发的主要文档来源。
- [LangGraph Python 官方文档](https://docs.langchain.com/oss/python/langgraph)
  **核心参考。** 覆盖 StateGraph、节点/边/条件边、Checkpointing、Human-in-the-Loop、子图。构建复杂 Agent 工作流的权威文档。
- [LangChain Academy](https://academy.langchain.com/)
  官方免费课程，覆盖 LangChain 和 LangGraph 的基础到高级。适合系统学习 + 面试准备。
- [LangChain GitHub (Python)](https://github.com/langchain-ai/langchain)
  源码和示例。适合深入理解 API 设计和查看最新变更。
- [LangGraph GitHub](https://github.com/langchain-ai/langgraph)
  源码和示例。适合学习 Graph API 的底层实现。

## 面试重点资源

- [LangGraph Use Cases: Agent Workflows](https://docs.langchain.com/oss/python/langgraph/workflows-agents)
  面试高频：如何用 LangGraph 构建 Tool-augmented Agent。包含完整可运行代码。
- [LangChain Multi-Agent: Subagents](https://docs.langchain.com/oss/python/langchain/multi-agent/subagents)
  面试高频：多 Agent 协作模式。子 Agent 作为工具的架构设计。
- [LangGraph Human-in-the-Loop](https://docs.langchain.com/oss/python/langgraph/thinking-in-langgraph)
  面试高频：interrupt() 机制、Command 恢复执行、MemorySaver。
